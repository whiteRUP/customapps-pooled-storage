OMV Mount Instructions

Mounting in OpenMediaVault:

  1. Start your pool from the "Storage Pools" section
  2. Note the mount protocol and port number
  3. In OMV, go to Storage → File Systems
  4. Click "Mount" and select the appropriate protocol:
    • WebDAV: Use "Remote Share" → WebDAV
    • NFS: Use "Remote Share" → NFS
    • SMB: Use "Remote Share" → SMB/CIFS
  5. Enter server address: 192.168.100.14
  6. Enter the port number displayed in the pool details
  7. Mount and access your pooled storage!
